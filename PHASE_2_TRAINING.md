## 1️⃣ **Tabla de Schulte** 👀

### ¿Qué es?
Clásico ejercicio ruso usado por lectores de élite para:
- Expandir visión periférica
- Entrenar fijación visual en el centro
- Aumentar velocidad de lectura visual

### Cómo Funciona

1. **Niveles 1-5**:
   - Nivel 1: 5×5 (25 números)
   - Nivel 2: 6×6 (36 números)
   - Nivel 3: 7×7 (49 números)
   - Nivel 4: 8×8 (64 números)
   - Nivel 5: 9×9 (81 números)

2. **Mecánica**:
   - Números aleatorios dispersos en la tabla
   - Haz click en 1, 2, 3... hasta el último número
   - Cronómetro mide tiempo de completación
   - Mantén vista fija en el CENTRO

3. **Adaptación**:
   - ✅ <30s + 100% accuracy → Siguiente nivel
   - ⚠️ >60s o <80% accuracy → Nivel anterior
   - ➡️ Entre estos → Mismo nivel

### Beneficios Científicos
- ✓ Amplía campo visual efectivo en 30-40%
- ✓ Mejora velocidad de escaneo de texto
- ✓ Reduce tiempo de fijación ocular
- ✓ Entrena control de movimientos oculares

### Resultado Esperado
Después de 2-3 semanas: **+50% en velocidad de lectura visual**

## 2️⃣ **N-Back Test** 🧠

### ¿Qué es?
Ejercicio de memoria de trabajo comprobado científicamente:
- Mejora comprensión y retención
- Aumenta capacidad de procesamiento mental
- Entrena concentración y atención

### Cómo Funciona

1. **Mecánica**:
   - Se muestra una letra (A, B, C, etc.)
   - Debes recordar la letra de hace **N pasos atrás**
   - Responde "Sí" o "No" en 2 segundos

2. **Niveles**:
   - **1-Back**: ¿Es igual a la anterior?
   - **2-Back**: ¿Es igual a la de hace 2 pasos?
   - **3-Back**: ¿Es igual a la de hace 3 pasos?

3. **Adaptación**:
   - ✅ >85% accuracy + <1s respuesta → Siguiente nivel
   - ⚠️ ≤50% accuracy → Nivel anterior
   - ➡️ Entre estos → Mismo nivel

### Ejemplo de Secuencia (2-Back)
```
Mostrado:  A   B   A   C   A   B
Posición:  1   2   3   4   5   6
Match:     N   N   Y   N   Y   N
           (A)    (A)    (A)
```

### Beneficios Científicos
- ✓ Entrena la corteza prefrontal (control ejecutivo)
- ✓ Mejora retención de información
- ✓ Aumenta velocidad de procesamiento
- ✓ Fortalece memoria de trabajo (+40% típico)

### Resultado Esperado
Después de 3-4 semanas: **+35% en comprensión de lectura**

## 🧠 Ciencia Detrás

### Neuroplasticidad
- El cerebro se adapta al entrenamiento
- Ejercicios regulares crean nuevas vías neurales
- 3-4 semanas: cambios observables

### Visión Periférica
- Schulte entrena SCN (Superior Colliculus Nucleus)
- Mejora detección de movimiento periférico
- Reduce necesidad de fijar vista en cada palabra

### Memoria de Trabajo
- N-Back fortalece corteza prefrontal
- Mejora "working memory capacity"
- Aumenta velocidad de procesamiento

## 💡 Tips de Éxito

### Para Schulte
1. **Mantén vista en el CENTRO** - La tabla es grande a propósito
2. **No busques activamente** - Detecta números periféricamente
3. **Aumenta gradualmente** - Un nivel por semana
4. **Practica constante** - La consistencia es clave

### Para N-Back
1. **Escucha/Ve activamente** - No digas números en voz alta
2. **Responde rápido** - Velocidad importa
3. **Mantén precisión** - Mejor 70% consistente que 100% errático
4. **Descansa entre sesiones** - El cerebro necesita consolidar

### General
1. **Sincroniza con lectura** - Haz ejercicios después de leer
2. **Mismo horario diario** - La consistencia crea hábito
3. **Tracking** - Revisa tu perfil regularmente
4. **Celebra pequeños logros** - Cada badge es progreso real
